# [RE.cipe](https://seriyspray.github.io/Re.cipe/)

> Your recipes. Your terminal.

A TUI-styled recipe management app. Organize recipes in folders, filter by ingredients, plan weekly meals, and build shopping lists — all in a minimal terminal aesthetic.

## Live Site

**[https://seriyspray.github.io/Re.cipe/](https://seriyspray.github.io/Re.cipe/)**

## Stack

- HTML
- CSS
- No frameworks. No build step. No dependencies.

## Project Structure

```
re.cipe/
├── index.html   — the entire page
├── styles.css   — all styling
├── .gitignore
└── README.md
```

## Local Development

Just open `index.html` in your browser. No server needed.
